## 问题分析

### 发现的问题

您提到的解析问题书籍清单中，质量检查确实遗漏了**章节缺失**问题。

**原因**：
- 原质量检查器将"章节编号不连续"列为**警告**（warning），而非**阻塞性问题**（issue）
- 导致这些书籍在报告中显示为"通过"或高分，但实际存在章节缺失

### 已修复

**修改文件**: `core/book_graph_quality_checker.py`

**修改内容**：
1. 将"章节缺失"升级为**阻塞性问题**
2. 添加到 `BLOCKING_ISSUES` 列表
3. 修改检查逻辑：从 `warnings` 改为 `issues`

```python
# 修复前
if discontinuity:
    warnings.append(f"章节编号不连续：缺失章节 {discontinuity}")

# 修复后
if discontinuity:
    issues.append(f"⛔ 章节缺失：缺失章节编号 {discontinuity}")
```
