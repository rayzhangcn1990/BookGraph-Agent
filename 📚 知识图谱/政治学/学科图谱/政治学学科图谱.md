## 三、学科发展脉络

### 古代目的论正义观（古希腊至中世纪）

**特点**：正义与德性、良善生活紧密相连，法律旨在培养公民品格，分配依据功绩和地位

**进化/消亡原因**：宗教战争和现代多元社会的兴起，使得统一的良善生活观念难以达成，导致对中立性的需求

**核心动力**：社会资源分配的不平等现实, 多元价值观之间的冲突与张力, 对个体权利与集体利益平衡的持续探索

---

### 现代程序正义观（18 世纪启蒙运动至今）

**特点**：正义优先于善，强调权利、自由和程序公平，政府在各种良善生活观念间保持中立

**进化/消亡原因**：过度中立导致公共话语贫瘠，公民德性萎缩，无法解决深层道德分歧（如堕胎、同性婚姻）

**核心动力**：社会资源分配的不平等现实, 多元价值观之间的冲突与张力, 对个体权利与集体利益平衡的持续探索

---

### 古典功利主义（18 世纪末至 19 世纪）

**特点**：边沁提出量化计算，密尔引入快乐的质量区分，强调自由作为长期福利的基础

**进化/消亡原因**：面临个体权利不可侵犯的直觉挑战，以及价值不可通约的批评

**核心动力**：对道德确定性和科学化的追求, 社会改革和立法效率的实用需求, 对宗教教条和直觉主义的反叛

---

### 现代福利经济学（20 世纪至今）

**特点**：转化为成本效益分析，广泛应用于公共政策，但回避了人际效用比较的难题

**进化/消亡原因**：无法解释分配正义问题，对贫富差距缺乏道德敏感度

**核心动力**：对道德确定性和科学化的追求, 社会改革和立法效率的实用需求, 对宗教教条和直觉主义的反叛

---

### 早期自由意志主义（19 世纪）

**特点**：斯宾塞等人提出，强调社会达尔文主义和自然权利，反对国家干预

**进化/消亡原因**：因缺乏对弱势群体的关怀和大萧条时期的市场失灵而暂时退潮

**核心动力**：对极权主义和政府过度干预的历史反思, 自由市场资本主义的意识形态支撑, 对个人自主性的高度重视

---

### 现代自由至上主义（20 世纪 70 年代至今）

**特点**：诺齐克、弗里德曼等人主导，通过哲学论证（如持有正义）和经济分析复兴，反对福利国家

**进化/消亡原因**：面临贫富差距扩大和社会团结瓦解的批评，但在科技新贵和自由市场倡导者中仍有影响力

**核心动力**：对极权主义和政府过度干预的历史反思, 自由市场资本主义的意识形态支撑, 对个人自主性的高度重视

---

### 古典德性论（古希腊）

**特点**：与政治共同体紧密绑定，德性即公民参与公共生活的能力

**进化/消亡原因**：基督教兴起将德性内化为灵魂救赎，现代国家兴起将道德私人化

**核心动力**：对道德相对主义和虚无主义的抵抗, 对社会凝聚力下降的担忧, 对人性完善和潜能实现的追求

---

### 当代共同体主义复兴（20 世纪 80 年代至今）

**特点**：批判自由主义中立性，呼吁回归公共善和公民责任

**进化/消亡原因**：面临多元社会如何避免压迫的质疑

**核心动力**：对道德相对主义和虚无主义的抵抗, 对社会凝聚力下降的担忧, 对人性完善和潜能实现的追求

---

### 康德批判时期（1785 年《道德形而上学基础》）

**特点**：正式提出绝对命令，区分自律与他律，确立义务论伦理学

**进化/消亡原因**：为了回应休谟对因果律和道德情感的怀疑论挑战

**核心动力**：理性能力的觉醒, 对欲望奴役的反思, 对道德普遍性的理性追求

---

### 社会契约论早期形态（17-18 世纪（霍布斯、洛克））

**特点**：基于历史或假想的真实同意，强调保护财产和生命

**进化/消亡原因**：无法解释未签署契约者的义务，且易受既有权力结构影响

**核心动力**：消除道德任意性因素的需要, 确保正义原则的不偏不倚, 对功利主义牺牲少数人利益的反思

---

### 罗尔斯的假想契约（1971 年《正义论》）

**特点**：引入无知之幕，将契约从历史事实转变为道德思想实验

**进化/消亡原因**：为了解决传统契约论无法保证实质公平的问题

**核心动力**：消除道德任意性因素的需要, 确保正义原则的不偏不倚, 对功利主义牺牲少数人利益的反思

---

### 自由至上主义分配（市场经济早期理论）

**特点**：完全由市场决定，强调形式机会平等，接受结果不平等

**进化/消亡原因**：忽视了起点不公和道德任意性，导致贫富悬殊

**核心动力**：对道德任意性的纠正, 社会团结与互惠的需要, 避免极端平等主义的负面后果

---

### 优绩主义分配（20 世纪中期）

**特点**：强调公平的机会平等，试图通过教育消除社会障碍

**进化/消亡原因**：仍无法消除自然天赋的道德任意性影响

**核心动力**：对道德任意性的纠正, 社会团结与互惠的需要, 避免极端平等主义的负面后果

---

### 罗尔斯差异原则（1971 年后）

**特点**：超越优绩主义，将天赋视为公共资产，关注最不利者

**进化/消亡原因**：认识到即使机会平等，自然运气仍导致不公

**核心动力**：对道德任意性的纠正, 社会团结与互惠的需要, 避免极端平等主义的负面后果

## 四、学科核心思想及底层逻辑

### 正义

**定义**：给予每个人其所应得的东西，涉及收入、财富、义务、权利、权力、机会、公共职务与荣誉的分配方式

**底层逻辑**：
```
前提假设：社会资源稀缺且人类需求多样，需要分配规则→推理链条：不同的分配原则（福利、自由、德性）对应不同的正义观，且彼此冲突→核心结论：正义没有单一的定义，而是通过公共辩论在竞争性的道德主张中不断被重构
```

**代表人物**：迈克尔·桑德尔

### 功利主义

**定义**：一种道德哲学，主张行为的正确性取决于其能否最大化整体幸福或福利，即'最大多数人的最大幸福'

**底层逻辑**：
```
前提假设：痛苦和快乐是主宰人类行为的主权者→推理链条：道德的目标是最大化净快乐，因此需计算行为后果→核心结论：正义的政策是能使社会总福利最大化的政策
```

**代表人物**：迈克尔·桑德尔

### 自由至上主义

**定义**：一种政治哲学，强调个人对自己及财产的绝对所有权，主张最小政府，反对强制再分配

**底层逻辑**：
```
前提假设：个人拥有自身，因此拥有劳动成果→推理链条：征税是强制劳动，再分配侵犯自我所有权→核心结论：政府仅应保护人身和财产安全，不应干预市场分配
```

**代表人物**：迈克尔·桑德尔

### 德性伦理

**定义**：一种伦理学进路，关注行为者的品格和动机，而非单纯的规则或后果，强调通过习惯培养优良品质

**底层逻辑**：
```
前提假设：人类有特定的功能或目的（Telos）→推理链条：德性是实现这一目的的优秀品质→核心结论：正义的社会应鼓励德性，法律应体现对良善生活的判断
```

**代表人物**：迈克尔·桑德尔

### 自律

**定义**：根据自己给自己确立的法则行动，而非受外在欲望或自然法则支配

**底层逻辑**：
```
前提：人拥有理性能力→推理：理性可自我立法→自我立法即自由→自由即自律→结论：自律是道德行为的必要条件
```

**代表人物**：迈克尔·桑德尔

## 六、代表书籍与阅读网络

### [[公正（第三版）]]
**作者**：迈克尔·桑德尔
**出版年份**：2009

**关联关系**：
- 与 [[正义论]]：本书主要对话和批判的对象，代表现代自由主义正义观，提供无知之幕和差异原则理论
- 与 [[自由主义与正义的局限]]：桑德尔博士论文，本书的学术理论基础，对罗尔斯自由主义的批判源头
- 与 [[金钱不能买什么]]：本书主题的延伸作品，更深入探讨市场道德界限
- 与 [[无政府、国家与乌托邦]]：自由至上主义的代表作，提供自我所有权和持有正义理论，与本书形成理论对立
- 与 [[道德与立法原理导论]]：功利主义奠基之作，提供快乐量化计算的理论框架
- 与 [[功利主义]]：密尔对边沁功利主义的修正，引入高级快乐与人性尊严概念
- 与 [[道德形而上学基础]]：康德义务论伦理学经典文本，提供绝对命令和自律概念
- 与 [[尼各马可伦理学]]：古典德性伦理学的源头，提供目的论正义观的理论依据
- 与 [[社会契约论]]：卢梭的公民义务理论，为反对公共服务商品化提供支持
- 与 [[理想国]]：西方正义理论的源头，柏拉图提出正义即各司其职
- 与 [[离开欧麦拉城的人]]：勒古恩短篇小说，作为功利主义批判的文学案例
- 与 [[刘擎西方现代思想讲义]]：序言作者作品，同样致力于思想普及，提供现代思想背景参照
